## **2️⃣ while 문 종료**

```java

public class Exam {

	public static void main(String[] args) {
		exit : while(true) {
			if(true) {
				System.out.println("hello");
				break exit;
			}
		}
		
	}

}
```

- while 앞에 라벨링의 이름을 붙여주고 종료할 때 라벨링 이름을 써주면 while 문 종료
